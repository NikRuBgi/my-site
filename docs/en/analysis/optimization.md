# Optimization Hints

!!! info "Analytics Plan"
    This feature requires the Analytics Plan.

## What are Optimization Hints?

Optimization hints are energy-efficiency recommendations linked to entity families. They highlight common improvement opportunities for a given equipment type and guide analysts toward corrective measures.

Hints are pre-defined suggestions — they do not automatically detect a condition, but rather serve as a structured checklist of known optimization strategies for each family type.

## Accessing Optimization Hints

Optimization hints are available in **Modeling → Families**, on the configuration page for each family. They can also be surfaced in dashboard widgets for review by technical teams.

## Using Hints in Practice

Optimization hints are typically reviewed during analysis sessions. When a hint is relevant:

1. Investigate the corresponding data points and rules in the Explorer or Query Page
2. If an issue is confirmed, create a [Task](../tasks/workflow.md) to track the corrective measure
3. Link the task to the affected entity for traceability
