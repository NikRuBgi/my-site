# Users & Permissions

## User Management

Users are managed under **Administration → Users**. Each user is assigned a role that controls which modules they can view and edit.

## Access Levels

Dybee uses role-based access control. Higher-privilege roles include access to:

- Building Administration (configuration, user management)
- Data Acquisition (BACnet point mapping, Dybee-Link deployment)
- Modeling (entity creation, family configuration)

Standard users have read and write access to operational modules (dashboards, explorer, tasks, reports) but cannot modify the building model or system configuration.

!!! tip
    Assign administration roles only to the analysts or engineers responsible for the building model. Energy managers typically operate with standard user access.
