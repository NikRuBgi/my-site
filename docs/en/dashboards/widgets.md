# Widget Reference

Widgets are organized into six categories. All widgets require a title and a width (1–4 grid cells). An optional hyperlink can be added to any widget.

## General

| Widget | Description |
|---|---|
| Image | Displays a building photo or custom image |
| Temperature | Live NASA weather data: outdoor temperature, humidity, wind speed. Requires building coordinates. |
| Notes | Free text block for instructions, context, or documentation |
| Carousel | Rotating image slideshow |

## Energy

Energy widgets require completed invoice entry and a baseline model in **Energy Monitoring → Measurement & Verification**.

| Widget | Description |
|---|---|
| Performance Tracking | Actual vs. predicted consumption per period |
| Savings Monitoring | Savings in %, CO₂, and currency vs. baseline |
| Comparative Analysis | Building performance vs. benchmarks |
| Cumulative Energy | 12-month rolling consumption trend |
| Energy Results Table | Comprehensive exportable summary table |

## Operation

!!! info "Analytics Plan"
    Operation widgets require acquired data points and modeling (Analytics Plan).

| Widget | Description |
|---|---|
| Graphics Module | Embeds a chart created in the Visualization module |
| Temporal Comparison | Overlays data from two time periods |
| Point Lists | Latest values with min/max/avg/total statistics |
| Explorer | Links to a saved point set in the Explorer module |
| KPI Pie Chart | Proportional breakdown of KPI values across entities |
| KPI Pyramid | Worst-performing entities for a selected KPI |
| Value Display | Single point value with trend indicator |
| Gauge | Current value or 24-hour average displayed as a gauge |
| Data Table | Hourly matrix with aggregation options |

## Management

| Widget | Description |
|---|---|
| Task Summary | Count of tasks by status |
| Task Tracking | Monthly rates for task creation, acceptance, and completion |
| Fault Detection | Auto-detected faults from the last 24 hours |
| Optimization Measures | Recommendations for technical teams |

## Module

| Widget | Description |
|---|---|
| Comparative Analysis | Actual performance vs. AI-generated reference model with acceptable limits |

## Other

| Widget | Description |
|---|---|
| Fault Detection Summary | Numerical fault count organized by priority level |

## Using Widgets from Other Users

The **Other Users** tab in the widget picker allows you to import and reuse widgets created by teammates. This is useful for standardizing views across team members on the same building.
