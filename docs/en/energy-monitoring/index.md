# Energy Monitoring — Overview

Energy monitoring is available on all plans and covers the full lifecycle of energy performance tracking: from raw invoice data to savings verification.

## What You Can Do

- Track energy consumption across multiple sources (electricity, gas, water, etc.)
- Build IPMVP-compliant baseline models for savings measurement
- Calculate and report energy savings, cost savings, and GHG reductions
- Analyze power demand peaks for tariff optimization
- Compare buildings across a portfolio

## Workflow

```
Building config → Energy sources → Invoice data → Baseline model → Savings analysis
```

## Sections

- [Energy Sources](sources.md) — set up energy types and units
- [Invoice Data](invoices.md) — enter consumption data manually, via Excel, or automatically
