# What is Dybee?

Dybee is a cloud-based platform for continuous building commissioning (CxC). It helps teams maintain optimal building performance over time by combining energy monitoring, system modeling, real-time data acquisition, and automated fault detection.

## Implementation Methodology

Dybee is structured around four implementation stages. How far you progress depends on your subscription plan.

| Stage | Description | Plan |
|---|---|---|
| 1 | Building configuration and user management | All plans |
| 2 | Energy monitoring — baseline modeling and intervention tracking | All plans |
| 3 | Digital twin modeling, Dybee-Link installation, and data acquisition | Analytics Plan |
| 4 | Fault detection, optimization analysis, and task management | Analytics Plan |

## Core Principle: Continuous Commissioning

Unlike a one-time optimization, Dybee is designed for ongoing monitoring. The platform supports iterative analysis cycles — detect a drift, create a corrective task, track resolution, repeat. This methodology is referred to throughout the documentation as **CxC** (continuous commissioning).
